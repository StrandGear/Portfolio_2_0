A side project at Spoonful: a visual novel–style local co-op exhibited in a museum. Multiple stations let student groups play simultaneously, each with a unique story path.

I developed a custom dialogue editor in Unity GUI, featuring multiple node parameters and tools to streamline the creation of branching dialogue trees. To achieve this, I implemented a peer-to-peer networking system: all devices on the local network could act as a host, and with one button press the system synchronized connections across up to five devices. Each station then automatically loaded a different character and storyline depending on the game state and player responses. The setup was designed to be simple and reliable so that museum staff could run the experience without requiring technical support.
